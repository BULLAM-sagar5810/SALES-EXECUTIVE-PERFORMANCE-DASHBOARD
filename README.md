# Sales Executive Performance Dashboard

## Project Overview

This project is an interactive Sales Executive Performance Dashboard developed using Microsoft Excel.

The dashboard helps business managers monitor sales performance, track target achievement, identify top and bottom performers, analyze regional performance, and make data-driven decisions.

## Business Problem

Organizations often struggle to monitor sales employee performance across different regions and evaluate target achievement efficiently.

This dashboard solves these challenges by providing:

* Total Sales Tracking
* Target Achievement Analysis
* Top & Bottom Performer Identification
* Regional Performance Comparison
* Interactive Filtering using Slicers
* VBA-Based Automation

## Tools Used

* Microsoft Excel
* Pivot Tables
* Pivot Charts
* Slicers
* VBA Macros
* Data Cleaning
* KPI Design

## Key KPIs

1. Total Sales
2. Target Achievement %
3. Away From Target %
4. Regional Performance

## Features

* Interactive Dashboard
* Region-Based Filtering
* Automated Slicer Connections using VBA
* Top 5 Performer Analysis
* Bottom 5 Performer Analysis
* Business Intelligence Reporting

## Business Insights

* Top-performing employees contribute significantly to total sales.
* Target achievement varies across executives.
* Regional performance differences reveal growth opportunities.
* Automation reduces manual reporting efforts.
